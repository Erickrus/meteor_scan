# meteor_scan
Scan through video for meteor 

# Installation
Install python packages based on requirements.txt
`pip3 install -r requirements.txt`

Also install `ffmpeg`
`apt install -y ffmpeg`

to run this program
`
python3 meteor_scan.py meteor01.mp4
`

Enjoy!
